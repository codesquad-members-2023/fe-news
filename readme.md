# 프로젝트 개요

`네이버 뉴스스탠드` 클론코딩 프로젝트입니다. `2023.4.3~2023.4.21`까지 3주간 진행되었습니다. React 등의 프레임워크 없이 `typescript` 만을 사용해 작성되었습니다. 이 프로젝트는 OOP를 중점적으로 학습하였습니다.
